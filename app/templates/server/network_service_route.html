route计算多个途经点之间的最短路径。

此操作的参数在以下参数表中定义:

参数 说明
stops

用于定义路径的停靠点。至少需要两个点。

简单结构: 

x1,y1; x2,y2; ...; xn, yn

示例：

121.433, 31.242; 121.45, 31.235

JSON响应语法

JSON结构:

{
  "lengths" : [ <length1>, <length2>, ..., <lengthN> ],
  "areas" : [ <area1>, <area2>, ..., <areaN> ]
}

示例:

[
  {
    "areas" :
    [
      2150.340979272913,
      2800449.740201426,
    ]
  },
  {
    "length" :
    [
      222.93213167603818,
      264.5927746444603,
    ]
  }
]
